# Frontend del dashboard

Este directorio contendrá la aplicación web del dashboard final.

## Objetivo

Implementar la experiencia interactiva del informe final usando D3.js y una organización clara del frontend.

## Estructura sugerida

```text
dashboard/frontend/
├── public/   # assets estáticos
└── src/      # código fuente del frontend
```

## Vistas objetivo

Según la arquitectura definida:

1. Repetición y desempeño
2. Dificultad por KC / ruta conceptual
3. Perfil de estudiantes

## Nota

El frontend debe consumir datasets ya preparados desde `dashboard/data/`.
